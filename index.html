AR Heatmaps

This is an Android application built using ARCore and Sceneform to display heatmaps in an AR space. The advantage of these heatmaps over traditional heatmaps (displayed in print or on a display) is that they can cover large spaces and be resized on demand. Key features of the application include the ability to add/display custom heatmaps, resize the heatmaps, and place them anywhere in the physical space. The source code can be found at: https://github.com/HeatmapDojo/ar-heatmap/tree/arcore/ARCore.

Mutual Aid Database Demo

This is a prototype for a website that connects people seeking aid to those who can provide it. Unlike traditional platforms for mutual aid (e.g. Facebook, social media, community groups) where the person who needs aid has to request help before the helper can respond, the Mutual Aid Database has helpers share ways they can help on a public forumn where people who need aid can contact them. This solves the timing problem where people seeking aid are often forced to wait long periods of time before they find someone who is willing to help. The source code can be found at https://github.com/wesleytchen/madb-demo and the prototype is hosted at https://wesleytchen.github.io/madb-demo.

Carbon Foodprint

This is a website I made alongside a couple of my friends at the UncommonHacks 2020 hackathon. It helps users understand the impact of their diet on the environment by calculating the carbon footprint of the foods they eat. Specific CO2 levels are tracked for each food within a database and the calculation of CO2 is based on how many kg of the food the user inputs. The source code can be found at: https://github.com/efraimdahl/UC-Hacks-2020 and the website can be run by downloading the source code.
