MyFoodprintPal

This is a website I made alongside a couple of my friends for the 2022 Design Challenge, which we were finalists in. It helps users understand the impact of their diet on the environment by calculating the carbon footprint and water usage of the foods they eat. Specific CO2 and water usage levels are tracked for each food within a database and the calculation of CO2 is based on how many kg of the food the user inputs. The source code can be found at https://github.com/efraimdahl/UC-Hacks-2020 and you can try out the site at https://wesleytchen.github.io/myfoodprintpal/.

Swappable Sentences

Swappable Sentences is a word processor add-on built by my friends and I over three months. This tool allows users to create alternative sentence options and swap them into their essay drafts with the click of a button. We used JavaScript, HTML, and CSS to build the application. Our project won the People's Choice Award at the 2021 Spring Tech Showcase and we placed second for UChicago Impact. Our presentation can be found here.

AR Heatmaps

This is an Android application built using ARCore and Sceneform to display heatmaps in an AR space. The advantage of these heatmaps over traditional heatmaps (displayed in print or on a display) is that they can cover large spaces and be resized on demand. Key features of the application include the ability to add/display custom heatmaps, resize the heatmaps, and place them anywhere in the physical space. The source code can be found at: https://github.com/HeatmapDojo/ar-heatmap/tree/arcore/ARCore
